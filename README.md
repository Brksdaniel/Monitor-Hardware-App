# 📱 Monitor de Hardware App

App Android que monitora seu PC em tempo real direto do celular!
Veja tensões, temperaturas e ventoinhas do seu PC onde estiver.

![Expo](https://img.shields.io/badge/feito%20com-Expo-000020)
![Supabase](https://img.shields.io/badge/banco-Supabase-3ECF8E)
![Android](https://img.shields.io/badge/plataforma-Android-3DDC84)

---

## ⚡ O que mostra?

- Tensões da placa-mãe, CPU e GPU
- Temperaturas de todos os sensores
- Velocidade das ventoinhas em RPM
- Atualização automática a cada 3 segundos

---

## 🔗 Como funciona?
```
PC (Monitor de Hardware XP)
        ↓
   Supabase (nuvem)
        ↓
  📱 App Android
```
Caso queiram baixar o apk >>> https://expo.dev/accounts/brkadaniel/projects/monitor-hardware-app/builds/b3db7e6a-3382-46cb-a37b-d4052cc8c2a7

---

## 🚀 Como usar

1. Instale o app **Expo Go** no celular
2. Clone o projeto e rode `npx expo start`
3. Escaneie o QR code com o Expo Go
4. Deixe o **Monitor de Hardware XP** rodando no PC
